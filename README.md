# React-Native-Github

这是一个用来查看GitHub最受欢迎与最热项目的App,它基于React Native支持Android和iOS双平台。
参考的是：<https://github.com/crazycodeboy/GitHubPopular>

## 技术栈

React Native + ES6 + AsyncStorage + Fetch + Native Moudules + 第三方库

	.
